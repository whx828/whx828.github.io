关于博客

从 2022 年开始，个人生活的简单记录。
之前尝试过 Next.js，一直到部署，跟着官方教程都比较顺利，但之后确实是过于高端了，完全不是给新手准备的。那之后我意识到我只是需要一个记录的地方，不必自己从头完成所有的工作。恰逢我对 Rust 产生了极大的兴趣，因此选择对 zine 做减法，建成了个人博客，并利用 Github Pages 让网站上线了，其实我还没搞懂没有服务器的情况下该怎么部署网站。
尽管到处都充满了「极 凑 简 合」的气息，我的博客之路还是开始了，经过一段时间的折腾，我发现想要在互联网上玩得开心，一定要

感兴趣的

Crafting Interpreters

这是一本带你写解释器的书，使用 Java 和 C 分别实现一门叫做 Lox 的语言的解释器，书里的操作令我眼花缭乱，作者不愧写了 10 多年，目前以龟速阅读中，对我的 Java 有很大提高，非常感谢作者。

The Missing Semester of Your CS Education

MIT 的一门对于编程世界里的「杂活」进行正规讲解的课程，里面对于 vim、git 等的解释是我见过最好的，非常适合我这样的初学者理解。

Amos 的博客

我非常喜欢的 Rust 博主，最近在看他的博文，很有收获。
