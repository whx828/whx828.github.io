关于我

这里有 2021 年的几篇旧文收录，以及从 2022 年开始，个人生活的简单记录。

我对编程比较感兴趣，但没有系统地学过；
我的专业是机械，程序员的主要来源之一（笑）；
我学过一点 Python，目前在学 Java 和 Rust，我更喜欢后者；
我尝试过一点前端开发，但过程不很美妙；
由于专业的关系，我对嵌入式算是耳濡目染，但没啥兴趣。

除了编程，我喜欢读书和听音乐，今年读过最有印象的一本书是《黑客与画家》；
我喜欢 BABYMETAL、椎名林檎、牛尾憲輔、Blizard、Dead End、伍佰和花伦；
我还不知道以后要做什么，平凡无趣的人生似乎也不错。

推荐

Crafting Interpreters

这是一本带你写解释器的书，使用 Java 和 C 分别实现一门叫做 Lox 的语言的解释器，书里的操作令我眼花缭乱，作者不愧写了 10 多年，目前以龟速阅读中，对我的 Java 有很大提高，非常感谢作者。

The Missing Semester of Your CS Education

MIT 的一门对于编程世界里的「杂活」进行正规讲解的课程，里面对于 Vim、git 等的解释是我见过最好的。

Amos 的博客

我非常喜欢的 Rust 博主，最近在看他的博文，很有收获。

九曲阑干

CSAPP 的讲解视频，非常棒，对自学者很有帮助，该有的都有。

黑客与画家

技术和生活。我们也许做不了思想者，但可以做思考者。

How to code and launch a tiny website.

本站的由来之一，非常感谢作者鼓励他的读者自己动手实现一个网站，简单又如何呢？至少这也是自己的创造。
